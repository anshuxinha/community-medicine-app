# Project: STROMA (Community Medicine)

## Overview
A React Native (Expo) application for community medicine students and practitioners. Features include a content library, video lessons, streak tracking, and a premium subscription model.

## Goals
- Provide high-quality medical education content.
- Support offline access to study materials.
- Gamify learning with streaks and progress tracking.
- Monetize via premium subscriptions with RevenueCat integration.

## Architecture
- **Frontend:** React Native (Expo)
- **Backend/DB:** Firebase (Auth, Firestore)
- **Payments:** RevenueCat (Google Play Store)
- **Navigation:** React Navigation (Stack, Tabs)
- **State:** AppContext (React Context)
